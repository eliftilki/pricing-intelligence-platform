# Pricing Intelligence Platform

Marketplace satıcıları için rakip analizi, talep tahmini, fiyat optimizasyonu ve SLM tabanlı açıklama üreten AI destekli pricing intelligence platformu.

## Services

- web: Next.js frontend
- api_service: Main backend API
- agent_service: LangGraph orchestration and pricing decision pipeline
- ml_service: Demand prediction model service
- slm_service: Self-hosted SLM explanation service
- data_ingestion_service: Marketplace data collectors

## Core Pipeline

Competitor Intelligence → Feature Engineering → ML Prediction → Optimization → Risk Control → Recommendation → SLM Explanation
